# Smart Attendance System 🚀

A web-based Smart Attendance System using Face Recognition and QR technology with role-based access.

## Features
- Face Attendance using OpenCV + LBPH
- QR Attendance
- Admin & Viewer Roles
- Attendance Dashboard
- Duplicate Attendance Prevention
- Multiple Student Detection
- Flask Web Application

## Technologies Used
- Python
- Flask
- OpenCV
- HTML
- CSS
- NumPy
- QR Code
- CSV

bash
python app.py
